# cpp-http-server

A ground-up HTTP server project written in C++20 using POSIX sockets.

## Overview

This repository is focused on understanding how an HTTP server works from the inside out. The project is being developed without HTTP frameworks in order to study the core mechanics of request handling, response generation, and server architecture at a lower level.

## Goals

The current goals of the project are to:

- build an HTTP server from scratch
- strengthen C++ design and implementation skills
- study HTTP server architecture in a structured way
- develop the system incrementally

## Current Status

The project is currently in the architecture and design phase. Documentation and design artifacts are being developed before implementation begins.

## Scope

Initial development is focused on the server side of HTTP and targets HTTP/1.1.

## Notes

This project is intentionally being built step by step. The design is expected to evolve as understanding of the system becomes more concrete.
